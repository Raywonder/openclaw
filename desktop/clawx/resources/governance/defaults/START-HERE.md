# Start Here

This build is preconfigured for immediate use.

## Quick start (no training required)
1. Open ClawX.
2. Add one provider key in Settings (OpenAI, OpenRouter, Ollama, or your server).
3. Start Gateway (auto-start is enabled by default in this build).
4. Open Chat and start working.

## If you want zero setup
- Keep default settings.
- Use bundled defaults in `governance/`.
- Skip advanced policy edits until needed.

## When you want advanced control
- Edit files under `~/.clawx/governance`.
- Your local changes are never overwritten by app updates.
- Admins can still enforce stricter rules from server-side policy.
